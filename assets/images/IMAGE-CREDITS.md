# Image Credits

This project currently uses local SVG demo placeholders for trip images and background images. They are temporary assets for the frontend course phase and do not use external image URLs.

Future real trip images should be stored locally in:

- `assets/images/trips/`
- `assets/images/backgrounds/`

Do not rely on external image URLs in the final frontend demo. For every real image added later, record the source, author or owner, license, and any required credit note in this file.
