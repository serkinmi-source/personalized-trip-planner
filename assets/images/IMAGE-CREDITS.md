# Image Credits and Replacement Plan

This project currently uses local SVG demo placeholders for trip images and background images. They are temporary assets for the frontend course phase and do not use external image URLs.

Future real images must be saved locally in:

- `assets/images/trips/`
- `assets/images/backgrounds/`

Do not hotlink external image URLs in the final demo. Keep image file names consistent with `js/data.js`, record the source, author or owner if available, and include license or permission notes for every real image. Use compressed images where possible, and prefer landscape images for trip cards and hero backgrounds.

## Manual Image Replacement Steps

1. Add image files to `assets/images/trips/` and `assets/images/backgrounds/`.
2. Verify filenames exactly match the checklist.
3. Run Prompt 9B to update paths from placeholder SVGs to real image files.
4. Test Home, Recommendations, Trip Details, and My Trips.
